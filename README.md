# 👑 GGOD FATHERR VCF Maker Bot

A premium Telegram bot for converting, merging, splitting, and renaming contact files.

## ⚡ Features

| Feature | Description |
|---|---|
| 📁 TXT → VCF | Convert phone number lists to VCF contacts |
| 📊 TXT/VCF → CSV | Export contacts to CSV |
| 📇 CSV → VCF | Import CSV as VCF contacts |
| 📄 VCF → TXT | Extract phone numbers from VCF |
| 💬 MSG → TXT | Save any message as a TXT file |
| 📝 Rename File | Rename any uploaded file |
| ✏️ Rename Contacts | Bulk rename VCF contact names with a prefix |
| 🧩 Merge VCFs | Merge multiple VCF files into one |
| 📚 Merge TXTs | Merge multiple TXT files into one |
| ✂️ Split File | Split a large VCF/TXT into smaller chunks |
| 🛡️ Navy Format | Apply admin navy numbering to VCF contacts |

## 🚀 Setup

1. Install dependencies:
   ```bash
   pip install python-telegram-bot openpyxl
   ```

2. Set your bot token in `bot.py`:
   ```python
   BOT_TOKEN = "your_token_here"
   ```

3. Run the bot:
   ```bash
   python bot.py
   ```
